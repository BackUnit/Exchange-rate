# Финансовые Калькуляторы

Приложение на Kotlin с использованием Jetpack Compose, предоставляющее набор финансовых инструментов.

## Функционал

### Калькуляторы
1. **Сложный процент** - расчет доходности инвестиций с учетом сложного процента
   - Настройка начальной суммы
   - Процентная ставка
   - Срок инвестирования
   - Частота начисления процентов (ежегодно, ежеквартально, ежемесячно, ежедневно)

2. **Кредитный калькулятор** - расчет параметров кредита
   - Сумма кредита
   - Процентная ставка
   - Срок кредита
   - Расчет ежемесячного платежа, общей выплаты и переплаты

3. **Конвертер валют** - конвертация валют по актуальным курсам
   - Подключение к API для получения актуальных курсов
   - Выбор валюты источника и цели
   - Автоматическое обновление курсов

### Интерфейс
- Темная тема оформления
- Красивые градиентные карточки для калькуляторов
- Плавные анимации переходов
- Выдвижное меню с навигацией
- Абстрактные иконки для каждого калькулятора

### Меню
- Настройки приложения
- Поддержка (FAQ, контакты, форма обратной связи)
- О приложении

## Технологии

- **Kotlin** - основной язык разработки
- **Jetpack Compose** - современный декларативный UI toolkit
- **Material 3** - дизайн-система
- **Coroutines** - асинхронное программирование
- **Navigation Compose** - навигация между экранами

## Сборка и запуск

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Запустите эмулятор или подключите устройство
4. Нажмите Run (Shift+F10)

## Требования

- Android SDK 24+
- Android Studio Hedgehog или новее
- Kotlin 1.9+
- Gradle 8.0+
